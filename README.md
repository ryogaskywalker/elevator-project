### 高層階用(23F-33~41F)
- 課題
 - 0930-1000あたりで混みすぎていて長時間並ぶこと
 - 列が中層階用のゲート通ろうとする人の邪魔にもなる
 - ただただ辛い
- 解決策
 - 奇数階用と偶数階用でエレベーターを分ける
  - 制御は変えられないのであくまで運用で変える
  - シミュレーションしてエビデンスを出す
   - ネット新人らしい戦い方で
   - 目指せメディア掲載

### 解決策
- エレベーターにルールを作る（張り紙程度）

- 具体的な要件は未定
